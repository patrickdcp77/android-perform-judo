# Documents

Ce dossier contient des documents de projet (images, schémas, notes, exports).

## Recommandations
- Images : `documents/images/`
- Schémas : `documents/schematics/`
- Exports (CSV, etc.) : `documents/exports/`

> Astuce : pour les ressources utilisées par l’app (icônes, images affichées dans l’UI), préfère `app/src/main/res/`.
